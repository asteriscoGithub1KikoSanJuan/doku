Mit position:fixed werden footer- und header-Element als feststehende Element definiert. Mittels bottom: 0; wird der footer 0 Pixel von der Unterkante des Darstellungsbereiches entfernt, also ganz unten positioniert. Der header wird mit top:0; ganz oben positioniert.

CSS-basierte Layouts

feststehender Header

header {

position: fixed;

padding: 10px;

top: 0;

left: 0;

right: 0;

text-align: center;

background: #c32e04;

border-bottom: 1px solid #d5d5d5;

}

feststehender Footer

footer {

position: fixed;

padding: 10px;

bottom: 0;

left: 0;

right: 0;

background: #c32e04;

border-color: #8a9da8;

}

HTML-Lesezeichen werden verwendet, um es den Lesern zu ermöglichen, auf bestimmte Teile einer Webseite zu springen. Lesezeichen können nützlich sein, wenn Ihre Webseite sehr lang ist. Um ein Lesezeichen zu erstellen, müssen Sie zuerst das Lesezeichen erstellen und dann einen Link hinzufügen. Wenn die Verknüpfung angeklickt wird, blättert die Seite zu dem Ort mit dem Lesezeichen. Erstellen Sie zuerst ein Lesezeichen mit dem id-Attribut: mit: id = "tips"> Dann fügen Sie einen Link zum Lesezeichen, von der gleichen Seite: href="#tips">

HTML-Tipps

HTML - Lesezeichen


Um ein Lesezeichen zu erstellen, müssen Sie zuerst das Lesezeichen erstellen und dann einen Link hinzufügen.

Erstellen Sie zuerst ein Lesezeichen mit dem id-Attribut:

<h2 id = "tips"> Nützliche Tipps Abschnitt </h2>

Dann fügen Sie einen Link zum Lesezeichen

<a href="#tips"> zum Abschnitt Nützliche Tipps </a>

Or, add a link to the bookmark ("Useful Tips Section"), from another page:

<a href="html_tips.html#tips">Visit the Useful Tips Section</a>

Dieser Absatz ist nur dafür da, damit Sie etwas zum Scrollen haben.
